# Как подключить Mini App к VK

## Шаг 1: Создание Mini App в VK

1. **Перейди в управление группой**
   - Открой свою группу VK
   - Нажми "Управление" → "Приложения"

2. **Создай новое приложение**
   - Нажми "Добавить приложение"
   - Выбери "Mini App"
   - Нажми "Создать приложение"

3. **Заполни данные**
   - Название: "Инвентарь сталкера"
   - Описание: "Управление инвентарём в Зоне"
   - Категория: "Игры"

4. **Получи app_id**
   - После создания появится app_id (например, 1234567)
   - Сохрани его — он понадобится!

---

## Шаг 2: Размещение на хостинге

Mini App должно быть доступно в интернете. Варианты:

### Вариант А: Vercel (бесплатно, рекомендую)

1. Зарегистрируйся на vercel.com
2. Установи Vercel CLI: `npm i -g vercel`
3. В папке `stalker_bot/inventory-app` выполни:
   ```
   vercel deploy
   ```
4. Получишь ссылку типа `https://stalker-inventory.vercel.app`

### Вариант Б: GitHub Pages

1. Загрузи файлы в GitHub репозиторий
2. В настройках включи GitHub Pages
3. Получишь ссылку вида `https://твой-ник.github.io/репозиторий`

### Вариант В: Свой сервер

Если есть сервер/хостинг:
- Загрузи файлы в папку `/var/www/inventory/`
- Настрой nginx
- Получишь ссылку `https://твой-домен.ru/`

---

## Шаг 3: Настройка Mini App

1. Вернись в VK → Управление группой → Приложения
2. Нажми на созданное приложение
3. В поле "Адрес" вставь ссылку на хостинге
   - Пример: `https://stalker-inventory.vercel.app`

4. В файле `app.js` замени строку:
   ```javascript
   // Тестовые данные (заменить на реальные с сервера)
   ```
   на реальный API запрос к боту

---

## Шаг 4: Запуск из бота

Добавь кнопку "Инвентарь" в клавиатуру бота:

```python
def create_main_keyboard() -> str:
    buttons = [
        [
            {
                "action": {
                    "type": "open_app",
                    "app_id": ТВОЙ_APP_ID,  # Полученный в Шаге 1
                    "owner_id": -ТВОЯ_ГРУППА,
                    "payload": ""
                },
                "label": "🎒 Инвентарь"
            }
        ],
        # ... остальные кнопки
    ]
    return create_keyboard(buttons)
```

## Подключение API бота

Чтобы инвентарь работал с реальными данными:

1. Создай API эндпоинт в боте:
   ```
   /api/inventory?vk_id=123456
   ```

2. В `app.js` замени тестовые данные:
   ```javascript
   async function loadInventory() {
       const response = await fetch('https://твой-домен.ru/api/inventory?vk_id=' + playerData.vk_id);
       const data = await response.json();
       playerData.inventory = data.inventory;
       playerData.money = data.money;
       playerData.health = data.health;
       // и т.д.
   }
   ```

## Частые проблемы

| Проблема | Решение |
|----------|---------|
| Приложение не открывается | Проверь ссылку в настройках Mini App |
| Данные не загружаются | Проверь консоль браузера (F12) |
| Ошибка VK Bridge | Добавь `https://unpkg.com/@vkontakte/vk-bridge/dist/vk-bridge.min.js` |
